SVG Reference

Elemento Resultado Código XML / Atributos Clave
<rect>

Rectángulos y cuadrados.

 
<rect x="10" y="10" width="80" height="80" 
      rx="10" fill="#3b82f6" />
<circle>

Círculos perfectos.

 
<circle cx="50" cy="50" r="40" 
        fill="#ef4444" />
<ellipse>

Óvalos con dos radios.

 
<ellipse cx="50" cy="50" 
         rx="45" ry="25" />
<line>

Segmentos rectos.

 
<line x1="10" y1="10" x2="90" y2="90" 
      stroke-width="5" />
<polyline>

Líneas quebradas conectadas.

 
<polyline points="10,90 30,40 
                  50,60 70,10..." />
<polygon>

Formas cerradas.

 
<polygon points="50,10 90,90 10,90" />
<path>

Curvas de Bézier complejas.

 
<path d="M10,80 C40,10 65,10 90,80" 
      fill="none" stroke-width="4" />
<text>

Texto vectorial.

 
<text x="50%" y="50%" 
      text-anchor="middle">AI Labs</text>
<defs> + Gradientes

Efectos visuales.

 
<linearGradient id="g">...</linearGradient>
<circle fill="url(#g)" />
<clipPath>

Recorte de formas.

 
<clipPath id="cp">...</clipPath>
<rect clip-path="url(#cp)" />
